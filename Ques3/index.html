Inline Styles

This paragraph has inline styles applied. This is why this text has the salmon color.

Code:

                        <p style="color: salmon">...</p>

Internal Stylesheet

This section has been styled using an internal stylesheet.

Code:

                        <style>
                            .internal {
                                background-color: #222222;
                                color: #ffffff;
                                padding: 10px;
                            }
            
                            .internal h2 {
                                color: #ff6314;
                            }
                        </style>

External Stylesheet

This section and some other parts of the document have been styled using an external stylesheet. The stylesheet is stored in the same directory with the name styles.css.

Code for import under head:

                        <link rel="stylesheet" href="./styles.css">

Code:

                        .external {
                            background-color: #2c2f33;
                            padding: 10px;
                            color: #ffffff;
                        }
                        
                        .external h2 {
                            color: #7289da;
                        }
                        
                        .external code, .external pre {
                            color: #99aab5
                        }
